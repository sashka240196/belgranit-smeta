# БЕЛГРАНИТ — Калькулятор сметы

Веб-калькулятор для расчёта стоимости памятников и благоустройства.

## 🌐 Открыть онлайн

**→ [belgranit-smeta.netlify.app](https://belgranit-smeta.netlify.app)**  
*(ссылка появится после деплоя на Netlify)*

---

## 🚀 Запуск локально

### Вариант 1 — Python (рекомендуется)
```bash
# Windows
start.bat

# Или напрямую
python server.py
```
Откройте браузер: **http://localhost:8000**

### Вариант 2 — Прямо из папки
Просто откройте `index.html` в браузере двойным кликом.  
*(Некоторые функции могут не работать из-за CORS)*

---

## 📋 Функции

- **Каталог товаров** — стелы, подставки, цветники, надгробки, столбы, художественные работы, полировка
- **Формы стел** — В-1 до В-219, Г-1 до Г-66 (с фото)
- **Гранитный калькулятор** — 7 типов гранита с автоматическими наценками по размерам
- **Нестандартные размеры** — ввод произвольных габаритов
- **Смета** — корзина с редактированием количества, метров и примечаний
- **Наценка / Скидка** — автопересчёт итога
- **Разбивка по категориям** — структура заказа в итогах
- **История смет** — сохранение до 20 смет в браузере
- **Печать** — компактная смета на A4
- **Экспорт** — копирование текста, Viber/Telegram/WhatsApp
- **Тёмная/светлая тема** — переключение
- **Два режима** — Оптовая и Производственная смета
- **Курс валюты** — ручной ввод, итог в $ и BYR
